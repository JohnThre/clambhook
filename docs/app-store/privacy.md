# ClambHook Privacy Policy

Last updated: 2026-06-03

ClambHook is a private iPhone VPN/proxy router with metadata inspection, distributed only through the Apple App Store. It creates a local VPN configuration so traffic can be routed according to the user's profiles and rules.

Supported v1 profile protocol identifiers are `clambback`, `openvpn`, `shadowsocks`, `tor`, `trojan`, and `wireguard`.

ClambHook does not sell, use, or disclose VPN traffic data to third parties. Profile data, proxy credentials, private keys, connection metadata, traffic logs, and diagnostics stay on the user's device unless the user explicitly exports them.

ClambHook v1 inspection is metadata-only. It records connection targets, routing decisions, byte counts, timing, and hop status for local review. It does not install a certificate authority, perform TLS MITM, store request or response bodies, export HAR files, or provide body-level redaction workflows on the iPhone App Store path.

Apple diagnostics may include crash and performance data if the user has enabled sharing diagnostics with developers through Apple. ClambHook does not include third-party analytics, advertising SDKs, or tracking SDKs.

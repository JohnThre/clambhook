# ClambHook Privacy Policy

Last updated: 2026-06-20

ClambHook for macOS is a direct-download network tool for Apple Silicon Macs,
distributed from jpfchang.org. It offers System Proxy mode for apps that honor
macOS HTTP, HTTPS, and SOCKS proxy settings, and Enhanced Mode, a privileged
daemon-backed utun mode for device-wide routing according to the user's
profiles and rules.

Supported public profile protocol identifiers are `clambback`, `openvpn`,
`shadowsocks`, `tor`, `trojan`, and `wireguard`.

ClambHook does not sell personal information. Profile data, proxy credentials,
private keys, connection metadata, traffic logs, and diagnostics stay on the
user's Mac unless the user explicitly exports them or sends them to support.

ClambHook can show connection targets, routing decisions, byte counts, timing,
hop status, profile state, rule state, and recent logs for local review. HTTPS
capture is opt-in, local, and bounded. Sensitive headers and configured query
parameters are redacted before captures are stored.

When a user activates a direct-sale license, jpfchang.org receives the license
key for verification, an email address if provided, a generated install ID,
display name, platform, architecture, app version, activation time, and
deactivation state. License keys are hashed before storage. Profile contents are
not uploaded for license activation.

ClambHook for macOS uses a USD 99.99 direct-sale macOS license: it includes one
year of feature updates, versions released during that year remain usable, it
covers up to 4 active Apple Silicon Macs, and it is transferable between
devices. Optional USD 8.99 paid feature updates extend the feature-update
window. Creem is the
default checkout provider, and NowPayments may be used when enabled. Payment
providers process payment details; jpfchang.org stores checkout IDs, order IDs,
product IDs, license state, and purchase timestamps needed to deliver and
support the license.

Apple diagnostics may include crash and performance data if the user has enabled
sharing diagnostics with developers through Apple. ClambHook does not include
third-party analytics, advertising SDKs, or tracking SDKs.

Support contact: clambhook@jpfchang.org

# Website Copy Notes

ClambHook for macOS is a direct-download network tool for Apple Silicon Macs,
distributed only from jpfchang.org. It offers System Proxy mode for apps that
honor macOS proxy settings and Enhanced Mode, a privileged daemon-backed utun
mode for device-wide routing.

Supported public profile protocol identifiers: `clambback`, `openvpn`,
`shadowsocks`, `tor`, `trojan`, and `wireguard`.

Privacy posture:

- ClambHook does not sell personal information.
- Profile data, connection metadata, logs, and diagnostics stay on the Mac
  unless the user exports or sends them.
- ClambHook can show local connection targets, routing decisions, byte counts,
  timing, hop status, profile state, rule state, and recent logs.
- HTTPS capture is opt-in, local, and bounded. Users should send only redacted
  examples to support.
- Apple diagnostics may include crash and performance data if the user has
  enabled sharing diagnostics with developers.
- ClambHook does not include third-party analytics, advertising SDKs, or
  tracking SDKs.
